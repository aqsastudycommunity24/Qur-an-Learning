# AQSA Study Community - Website Pembelajaran Al-Qur'an

## Deskripsi

AQSA Study Community adalah platform pembelajaran Al-Qur'an modern yang menyediakan berbagai program pembelajaran dengan metode yang efektif dan terjangkau. Website ini menampilkan informasi lengkap tentang komunitas dan program-program pembelajaran yang kami tawarkan.

## Fitur Utama

### 1. **Halaman Beranda (Homepage)**
- Tampilan hero yang menarik dengan call-to-action
- Informasi ringkas tentang AQSA Study Community
- Navigasi yang user-friendly

### 2. **Tentang Kami (About Section)**
- Misi dan visi komunitas
- Deskripsi tentang komunitas kami
- Cards yang informatif dan menarik

### 3. **Program Pembelajaran (Learning Materials)**
Website menyediakan 6 program pembelajaran utama:

#### a. **Tajweed Al-Qur'an**
- Cara membaca Al-Qur'an dengan baik dan benar
- Kaidah-kaidah tajweed
- Tips pembelajaran efektif

#### b. **Tafsir Al-Qur'an**
- Penjelasan dan interpretasi Al-Qur'an
- Jenis-jenis tafsir
- Tafsir-tafsir terkenal dan terpercaya

#### c. **Hafalan Al-Qur'an**
- Program hafalan dengan metode sistematis
- Tahapan pembelajaran hafalan
- Syarat dan ketentuan program

#### d. **Fiqih Al-Qur'an**
- Hukum-hukum Islam dari Al-Qur'an
- Cabang-cabang fiqih
- Aplikasi praktis hukum Islam

#### e. **Bahasa Arab Al-Qur'an**
- Pembelajaran bahasa Arab khusus Al-Qur'an
- Kosakata dan tata bahasa
- Berbagai level kesulitan

#### f. **Adab dalam Al-Qur'an**
- Etika dan moral dari Al-Qur'an
- Nilai-nilai Qur'ani praktis
- Pengembangan karakter Islami

### 4. **Kontak (Contact Section)**
- Nomor kontak pengajar
- Formulir kontak online
- Informasi lokasi

## Teknologi yang Digunakan

- **HTML5**: Struktur halaman web
- **CSS3**: Styling dan desain responsif
- **JavaScript**: Interaktivitas dan animasi
- **Font Awesome**: Icon library
- **Google Fonts**: Typography profesional

## Fitur Responsif

Website ini dirancang untuk responsif di semua perangkat:
- Desktop (1200px ke atas)
- Tablet (768px - 1200px)
- Mobile (480px - 768px)
- Small Mobile (di bawah 480px)

## Instruksi Penggunaan

1. Download atau clone repository ini
2. Buka file `index.html` di browser Anda
3. Navigasi menggunakan menu untuk melihat program pembelajaran
4. Isi form kontak untuk mengirim pesan
